# GdocUniverse
Science Simulations

A collection of free interactive science simulations for educational purposes.

## Available Simulations

### 1. Gravity Sim
An interactive gravity physics simulation application that allows users to explore gravitational physics and planetary orbits.

**Features:**
- Interactive sliders to control object mass, planet mass, distance, and velocity
- Real-time visualization of gravitational forces
- Educational tool for learning physics
- Built with React and TypeScript

**Try it now:** [Open Gravity Sim](https://ai.studio/apps/drive/1_3w2iECDvLcl4fgBlo_dTOeYOBy7ljjj)

## Getting Started

Each simulation is available as an interactive web application. Click on the "Try it now" link for any simulation to open it in your browser.
